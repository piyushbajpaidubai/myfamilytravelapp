# ✈️ My Family Travel App

A React-based Travel Itinerary Planner. Plan your trips with ease — organize schedules, track expenses, manage packing lists, and save locations all in one place.

## Features
- 📅 **Schedule** — Day-by-day trip planning with activities and times
- 💰 **Budget** — Track expenses by category with running totals  
- 🎒 **Packing** — Manage packing lists with categories and check-off
- 📍 **Locations** — Save and organize trip locations

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Built With
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
